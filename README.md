# EcoSweep-Robotics
<img width="1536" height="1024" alt="EcoSweep_Banner" src="https://github.com/user-attachments/assets/f818c693-0b80-45c7-b7e8-9caf66a76bd7" />

Ecosweep is a robotics club that focuses on creating sustainable robots capable of cleaning the environment. This year, the members have been working on robot prototypes that can collect small trash like plastic bottles in the grass and parks like the Forest Park.

In December 2025, the club was able to finish their first prototype and presented it to the school assembly, a great milestone for the club. After that, the members have shifted their focus to create a more sophisticated robot using professional robotic components.

The members of the robotics club have learned much about robotics design, coding, circuitry, leadership, as well as how to help others in the process. EcoSweep is just the beginning of a long journey, and the club is looking forward to creating more impactful projects in the future
